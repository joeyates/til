# Key

The `C-` prefix means to press the `Crtl` key, e.g. 'C-X' means `Ctrl+X`
`M-` means `Meta`, i.e. `Alt`

# Program control

`C-X C-C` - exit

# Commands

`M-x` - execute command
`C-g` - cancel

# Buffers

`C-x b` - switch to buffer
`C-x C-b` - list buffers

# Help

`C-h ?` - help on help
`C-h k` - describe keybinding
`C-h f` - describe function
`C-h v` - describe variable
