# Global Packages

Install

```sh
npm install -g {{package}}
```

```sh
npm uninstall -g {{package}}
```

List

Indicate installation path and libraries installed:

```sh
$ npm list -g
```

Info:

* `npm root -g` - Path to global node_modules directory

# Local Development


```
npm install {{PATH}}
```
